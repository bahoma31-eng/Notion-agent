# تعليمات وكيل Copilot — Notion-agent

## 🔄 قواعد Auto PR

- بعد الانتهاء من أي مهمة، افتح Pull Request **فوراً** بدون انتظار
- **لا تفتح أبداً PR كـ Draft** — يجب أن يكون جاهزاً للمراجعة مباشرة `--no-draft`
- عنوان الـ PR يطابق عنوان الـ issue تماماً
- أضف `Closes #ISSUE_NUMBER` في وصف الـ PR دائماً لإغلاق الـ issue تلقائياً

## ⚡ قواعد Auto-merge

- بعد فتح الـ PR مباشرة، فعّل Auto-merge بالأمر: `gh pr merge --auto --squash "$PR_URL"`
- لا تنتظر موافقة يدوية للدمج
- إذا كان الدمج متعذراً، أبلغ بتعليق على الـ PR بالسبب

## 🛠️ قواعد عامة

- اللغة الافتراضية: Python
- جميع التعليقات في الكود بالعربية
- احفظ الملفات الناتجة دائماً في مجلد `plans/`
- استخدم Secrets فقط للبيانات الحساسة، لا تكتبها مباشرة في الكود
- لا تعدّل ملفات `.github/workflows` إلا إذا طُلب منك ذلك صراحة
